# DNF Character Vision Analyzer (AI Demo)

던전앤파이터 캐릭터의 이미지와 직업 정보를 기반으로  
**Google Gemini Vision LLM**이 전투 스타일을 분석해주는 데모 웹사이트입니다.

## 🚀 Features
- 던파 Neople Open API로 캐릭터 정보/이미지 자동 가져오기
- Gemini Vision으로 캐릭터 외형 분석
- 전투 스타일/성향을 자연스러운 한국어로 설명
- Next.js 기반의 모던 웹 UI
- TailwindCSS로 만든 다크 UI

## 🛠️ Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS v4
- Neople OpenAPI (DNF OFFICIAL)
- Google Gemini 1.5/2.0 Vision API
- Serverless API Routes
